# Contao Config Boilerplate

Dieses Verzeichnis enthält eine Basis-`config.yml` für Contao-Projekte.

## Inhalt

- `config.yml` - Standard Contao-Konfiguration mit:
  - `adminEmail` aus `.env`
  - Standard-Datums-/Zeitformate
  - Bildgrößen & Limits
  - Mailer-Transport "kontakt" über `MAIL_USER`

## Hinweis

Die Contao-`localconfig.php` sollte nicht ins Repo. Sie wird automatisch vom System generiert.
